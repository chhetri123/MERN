Lecture 02: Introduction to HTML

HTML (Hypertext Markup Language) is a standard markup language for creating web pages. It defines the structure of a document, including its title, headings, paragraphs, images, links, and more.

HTML is used to create both static and dynamic web pages. Static web pages are created by writing HTML code directly into a text file, while dynamic web pages use server-side programming languages (such as PHP, Python, or Node.js) to generate content based on user input or data from databases.
