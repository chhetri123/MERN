3.9. Introduction to Frames

Frames are used to divide the browser window into multiple sections, each of which can load a separate HTML document.

3.10. Creating Frames

Example of a frameset (not recommended for use in modern HTML):

<frameset cols="50%, 50%">
    <frame src="frame1.html">
    <frame src="frame2.html">
</frameset>

3.11. Nested Frames

Example of nested frames (not recommended for use in modern HTML):

<frameset rows="50%, 50%">
    <frame src="frame1.html">
    <frameset cols="50%, 50%">
        <frame src="frame2.html">
        <frame src="frame3.html">
    </frameset>
</frameset>
